User CRUD Application

A CRUD (Create, Read, Update, Delete) application built using Node.js, Express.js, MongoDB, and Mongoose.

Features

Create User
Read Users
Update User
Delete User
Technologies Used

Node.js
Express.js
MongoDB
Mongoose
Thunder Client
API Endpoints

Create User

POST /users

Get Users

GET /users

Update User

PUT /users/:id

Delete User

DELETE /users/:id

Database

MongoDB database named "userdb" is used to store user information.
